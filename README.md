**Technologies Used**
ASP.NET Core
C#
HTML / CSS / JavaScript
Visual Studio (IDE)

**How to Run the Project Locally**
To run this project on your local machine, please follow these steps:
Install Visual Studio (recommended version: 2022 or later)
Make sure the following workloads are installed:
ASP.NET and web development
Clone this repository:
git clone https://github.com/your-username/your-repo.git
Open the solution file (.sln) using Visual Studio
Restore NuGet packages (if not done automatically)
Press F5 or click Run to start the application
